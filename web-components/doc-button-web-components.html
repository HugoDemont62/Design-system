Bouton (web components)

Trois variants (Primary, Secondary, Tertiary), deux tailles. Élément personnalisé <ds-button> qui rend un <button> ou un <a> stylés comme la doc classes.

Variants

Primary Label
Secondary Label
Tertiary Label

Tailles

Default Primary Secondary Tertiary
Small Primary Secondary Tertiary

États

Tokens exposés

Identiques à la version par classes : les mêmes variables CSS du fichier compilé s’appliquent au nœud interne.

Token SASSRôleValeur par défaut
$btn-primary-colorCouleur texte/bordure Primary→ $color-brand
$btn-primary-background-hoverFond hover Primary→ $color-brand-100
$btn-primary-background-activeFond active Primary→ $color-brand-200
$btn-secondary-backgroundFond Secondary→ $color-brand
$btn-secondary-colorTexte Secondary→ $color-on-brand
$btn-tertiary-bgFond Tertiary→ $color-neutral-800
$btn-sm-font-sizeTaille texte sm→ $font-size-md
$btn-sm-padding-vPadding vertical sm→ $space-2
$btn-sm-padding-hPadding horizontal sm→ $space-6

Utilisation

Inclure web-components/ds-button.js puis utiliser <ds-button> (éléments personnalisés) :

<script src="web-components/ds-button.js" defer></script>

<ds-button variant="primary">Label</ds-button>
<ds-button variant="primary" size="sm">Label sm</ds-button>
<ds-button variant="secondary" disabled>Désactivé</ds-button>
<ds-button variant="tertiary" href="...">Lien bouton</ds-button>

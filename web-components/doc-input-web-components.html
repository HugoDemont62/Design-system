Tokens exposés

Identiques à la version par classes (styles sur le <label> et l’<input> internes).

Token SASSRôleValeur par défaut
$input-backgroundFond du champ→ $color-surface
$input-border-colorBordure par défaut→ $color-brand-200
$input-border-color-focusBordure focus→ $color-brand-400
$input-border-color-errorBordure erreur→ $color-danger
$input-text-colorCouleur texte saisi→ $color-text-primary
$input-placeholder-colorCouleur placeholder→ $color-neutral-500
$input-helper-colorCouleur texte d'aide→ $color-text-secondary
$input-helper-color-errorCouleur texte d'aide erreur→ $color-danger

Utilisation

Inclure web-components/ds-input.js puis déclarer les attributs (voir éléments personnalisés) :

<script src="web-components/ds-input.js" defer></script>

<ds-input
  label="Email"
  helper="Texte d'aide"
  placeholder="vous@exemple.com"
  field-id="email"
  type="text"
  required
></ds-input>

<ds-input
  type="password"
  label="Mot de passe"
  helper="8 caractères minimum"
  field-id="pwd"
  required
  minlength="8"
></ds-input>

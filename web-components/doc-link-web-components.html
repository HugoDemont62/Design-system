Lien (web components)

Deux variants. Le secondary est prévu pour les fonds sombres. Élément <ds-link> : même rendu que la doc classes.

Variants

Primary Lien principal
Secondary Lien secondaire

États

Hover/Focus Survolez-moi
Visited Lien visité
Désactivé (optionnel) Lien inactif

Tokens exposés

Identiques à la version par classes : styles appliqués sur le <a> interne.

Token SASSRôleValeur par défaut
$link-primary-colorCouleur texte Primary→ $color-brand
$link-primary-color-activeCouleur active Primary→ $color-brand-100
$link-secondary-colorCouleur texte Secondary→ $color-neutral-100
$link-secondary-color-visitedCouleur visited Secondary#FD81F0
$link-secondary-color-activeCouleur active Secondary→ $color-neutral-0

Utilisation

Inclure web-components/ds-link.js puis utiliser <ds-link> (éléments personnalisés) :

<script src="web-components/ds-link.js" defer></script>

<ds-link href="..." variant="primary">Lien principal</ds-link>
<ds-link href="..." variant="secondary">Lien secondaire</ds-link>
<ds-link href="..." target="_blank" rel="noopener noreferrer">Externe</ds-link>

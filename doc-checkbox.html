Checkbox

Quatre états : unchecked, checked, indeterminate, disabled. Le composant Toggle est un modificateur CSS de la checkbox.

États Checkbox

Toggle (modificateur)

Tokens exposés

Token SASSRôleValeur par défaut
$checkbox-sizeTaille de la case24px
$checkbox-borderBordure unchecked→ $color-brand-200
$checkbox-border-checkedBordure checked/indeterminate→ $color-brand
$checkbox-bgFond unchecked→ $color-surface
$checkbox-bg-checkedFond checked/indeterminate→ $color-brand
$checkbox-checkmarkCouleur du crochet→ $color-on-brand
$checkbox-disabled-borderBordure disabled→ $color-neutral-300
$checkbox-disabled-bgFond disabled→ $color-neutral-100
$checkbox-disabled-fgCrochet + label disabled→ $color-text-tertiary

Utilisation

<!-- Checkbox -->
<label class="checkbox">
  <input type="checkbox">
  <span class="checkbox__box"></span>
  <span class="checkbox__label">Label</span>
</label>

<!-- Indeterminate : ajouter data-indeterminate, JS le gère -->
<input type="checkbox" data-indeterminate>

<!-- Toggle -->
<label class="toggle">
  <input type="checkbox">
  <span class="toggle__track"></span>
  <span class="toggle__label">Label</span>
</label>
